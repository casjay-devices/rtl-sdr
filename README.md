# rtl-sdr
### Installation

```bash
sudo apt-get install libtool libusb-1.0-0-dev librtlsdr-dev rtl-sdr build-essential autoconf cmake pkg-config rtl-sdr libusb-dev
sudo git clone https://github.com/merbanan/rtl_433 /usr/local/src/rtl_433 && cd /usr/local/src/rtl_433

mkdir build && cd build && cmake .. && make && make install
```
